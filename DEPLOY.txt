SERVIQ DEPLOYMENT TO RENDER

STEP 1: PUSH TO GITHUB
  cd C:\Users\jerom\Downloads\ServiQ
  git init
  git add .
  git commit -m "Deploy to Render"
  git remote add origin https://github.com/YOUR_USERNAME/serviq.git
  git push -u origin main

STEP 2: CREATE RENDER SERVICE
  1. Go to https://dashboard.render.com
  2. Click "New +" -> "Web Service"
  3. Connect GitHub repository
  4. Select "serviq"
  5. Environment: Docker
  6. Click "Create Web Service"

STEP 3: SET ENVIRONMENT VARIABLES
  APP_NAME=ServiQ
  APP_ENV=production
  APP_KEY=Click Generate
  APP_DEBUG=false
  APP_URL=Your Render URL
  DB_CONNECTION=sqlite
  SESSION_DRIVER=database
  LOG_LEVEL=error

STEP 4: CREATE ADMIN USER (After deployment, in Render Shell)
  php artisan tinker

  $admin = new App\Models\User();
  $admin->name = 'Admin';
  $admin->email = 'admin@serviq.com';
  $admin->password = bcrypt('admin123');
  $admin->is_admin = true;
  $admin->email_verified_at = now();
  $admin->save();
  exit

STEP 5: TEST
  Visit https://YOUR-APP.onrender.com/admin/login
  Login: admin@serviq.com / admin123

TOTAL TIME: 20 minutes
